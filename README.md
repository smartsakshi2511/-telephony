# Telephony CRM & Softphone

A cloud telephony and CRM dashboard with VoIP-based softphone, campaign management, reporting, and admin controls.

---

## Features
- VoIP-based web softphone with click-to-dial
- Admin and agent dashboard
- Campaign and lead management
- Call recording and reporting
- Multi-language support
- Alerts, reminders, and notifications
- softphone integrated

---

## Tech Stack
- Frontend: React.js, HTML, CSS, SCSS
- Backend: Node.js, Express.js
- Database: MySQL
- VoIP Integration: Asterisk
- Other Libraries: Redux, Chart.js, SipJS
